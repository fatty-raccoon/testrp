1. Setup Philosophy

Offline first. Think before you connect. Never trust defaults. Privacy begins the moment you power on.

2. App Hygiene

  • Install apps from trusted sources only (e.g. F-Droid, APKMirror verified)
  • Use open-source alternatives where possible
  • Restrict permissions: No location, no mic, no camera unless essential

3. Firewalls & Filters

  • Use RethinkDNS or NetGuard to block trackers
  • Pair with NextDNS custom blocklists
  • Disable background data for all apps

4. App Profiles

  • Use Shelter or Island to sandbox apps
  • Clone risky apps and isolate their permissions
  • Switch profiles when handling sensitive tasks

5. Hardening the OS

  • Disable unnecessary system apps and bloatware
  • Use De-Googled OS if possible (Graphene, Calyx)
  • Keep airplane mode on when idle
